# No Bliggity - No Blog:  Tech Blog

## Description:
This is an application built to allow user to publish posts/blogs/articles along with any commentary.  

## Technologies Used
- Express.js
- Handlebars.js
- Sequelize
- MySQL2
- bcrypt
- dotenv
- express-sessions
